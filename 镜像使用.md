## 列出镜像列表

```shell
docker images
```

<span id="jump2"></span>

## 查找镜像

```shell
docker search 要找的镜像类别
```

---

<span id="jump3"></span>

## 拖取镜像

```shell
docker pull 镜像名
```

---

<span id="jump4"></span>

## 删除镜像

```shell
docker rmi 镜像名
```
